**PR Procedure**
**Coder Tasks**
- [ ] Coder merges main into feature branch.
- [ ] Coder reviews their own code as below.
- [ ] Coder creates PR.
- [ ] Coder assigns reviewer.

**Review Code**
- [ ] Reviewer merges main into feature branch.
- [ ] Code is functional (run the code) and performs as it should.
- [ ] Look through files changed for anything weird/not intended.
- [ ] Comments are sufficient and not overkill.
- [ ] Typos, spellcheck, professional.
- [ ] Reviewer sends back review.
      
**Edits**
- [ ] Coder fixes any issues.
- [ ] Reviewer approves.
- [ ] Reviewer assigns Michael.
      
**Michael Review**
- [ ] Michael reviews and sends back review.
- [ ] Coder fixes any issues.
- [ ] Michael approves.
- [ ] Michael merges.
